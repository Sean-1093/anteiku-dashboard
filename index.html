Connect to Google Sheets

Follow these 5 steps to connect your Anteiku data. Takes ~5 minutes.

1

Upload the Excel file to Google Sheets

Open drive.google.com → Click + New → File upload → Select Anteiku_Cafe_Database.xlsx → Once uploaded, right-click it → Open with → Google Sheets. It will convert automatically.

2

Make the Sheet public (read-only)

In your Google Sheet → click Share (top right) → click "Change to anyone with the link" → Set role to Viewer → Click Copy link → Click Done.

3

Get your Spreadsheet ID

From the link you copied, extract the ID between /d/ and /edit.
Example: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA.../edit
Your ID is the bold part.

4

Get a free Google API Key

Go to console.cloud.google.com → Create a project (or use existing) → Search "Sheets API" → Enable it → Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key.
